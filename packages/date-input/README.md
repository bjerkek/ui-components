# @ui-components/date-input

## Install
```
npm install --save @ui-components/date-input
```

## Usage
```
<ui-date-input></ui-date-input>
```

|  Props | Observable | Default | Description |
| --- | --- | --- | --- |

| locale | false | no | Supports no and en |
